## Quick Reference

### File Naming Convention

```
MF_{Bank}_{##}_{PatchName}
```

Examples:
- `MF_LoFi_Southern_01_SwampBass`
- `MF_Blues_Jazz_02_MidnightRhodes`
- `MF_BoomBap_04_BrassStab`
- `MF_DrumsFX_01_SynthKick`

### Patch Roles

| Role | Description | Count |
|------|-------------|-------|
| **Bass** | Low-end foundation | 5 |
| **Lead** | Melodic foreground | 7 |
| **Chords** | Harmonic support | 3 |
| **Texture** | Atmosphere/pads/noise | 6 |
| **Stab** | Short, punchy melodic hits | 1 |
| **Drums** | Kick, snare, hi-hat | 3 |
| **FX** | Risers, drops, transitions | 3 |

### Sampling Rule

**Always sample at C3** — see [MPC Sampling Guide](MPC-SAMPLING-GUIDE.md)
